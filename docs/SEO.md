## 4. Oportunidades de Nicho Priorizadas

### Nicho 1 — SPDA para Usinas Fotovoltaicas
- **Estados:** MG, ES, RJ, SP (MG é líder nacional em usinas solares)
- **Keywords âncora:** `laudo spda para usina solar minas gerais`, `laudo spda usina fotovoltaica espírito santo`
- **Concorrência:** Baixa/Média — poucos players B2B posicionados
- **Rota sugerida:** `/spda-para-raios#usinas-fotovoltaicas`

### Nicho 2 — SPDA para Torres de Telecomunicações
- **Estados:** RJ, SP, MG, ES
- **Keywords âncora:** `spda para torre de telecomunicações são paulo`, `laudo spda torre telecom rio de janeiro`
- **Concorrência:** Baixa/Média — generalistas dominam, sem especialistas visíveis
- **Diferencial Central:** carteira com Claro e Embratel como prova social
- **Rota sugerida:** `/spda-para-raios#torres-telecom`

### Nicho 3 — Licenciamento Ambiental para Postos de Combustível
- **Estados:** RJ, MG, ES, SP
- **Keywords âncora:** `licenciamento ambiental posto de combustível rj`, `licenciamento ambiental posto de combustível minas gerais`
- **Base legal:** Resolução CONAMA 273
- **Concorrência:** Média/Alta, mas poucos com abrangência em 4 estados simultaneamente
- **Rota sugerida:** `/licenciamento-ambiental#postos-combustivel`

### Nicho 4 — Pacote Integrado AVCB + SPDA para Galpões Logísticos e Indústrias
- **Estados:** RJ, SP, MG, ES
- **Keywords âncora:** `avcb e laudo spda para galpão logístico rj`, `regularização completa galpão logístico rj`
- **Concorrência:** Alta em termos genéricos, Baixa/Média com combinação serviço+setor+estado
- **Rota sugerida:** `/avcb-corpo-de-bombeiros#galpaos-logisticos`

### Nicho 5 — Regularização Completa para Agroindústria / Agronegócio
- **Estados:** MG, ES, RJ, SP
- **Keywords âncora:** `avcb e licenciamento ambiental para agroindústria minas gerais`, `regularização completa agroindústria rj`
- **Concorrência:** Baixa — consultorias ambientais não cobrem AVCB e SPDA
- **Diferencial Central:** posicionamento "tudo em um só lugar"
- **Rota sugerida:** `/licenciamento-ambiental#agroindustria`

## 7. Tipos de Conteúdo Recomendados (Blog / FAQ)

1. **Guias completos por tema + estado**
   - Exemplo: "AVCB RJ: guia completo para obtenção"
   - Exemplo: "Licenciamento Ambiental ES: passo a passo pelo IEMA"

2. **Artigos por problema + setor**
   - Exemplo: "Como emitir laudo SPDA para torres de telecom conforme NBR 5419"
   - Exemplo: "Erros que reprovam AVCB em usinas solares"

3. **FAQs estruturadas com schema FAQPage**
   - Exemplo: "Quanto tempo demora para sair o AVCB RJ?"
   - Exemplo: "Quem pode assinar um laudo de SPDA?"
   - Implementar via JSON-LD `FAQPage` em cada `page.tsx`
